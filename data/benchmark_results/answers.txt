# LRU Cache Design

An LRU (Least Recently Used) cache is a data structure that stores key-value pairs and evicts the least recently used item when the cache reaches its capacity. It supports `get` and `put` operations in O(1) average time using a combination of a hash map and a doubly linked list.

## Data Structures Used

1. **Hash Map (`dict`)**: Provides O(1) average time for key lookups.
2. **Doubly Linked List**: Maintains the order of access, with:
   - **Head**: Most recently used item
   - **Tail**: Least recently used item

## Python Implementation

```python
class Node:
    def __init__(self, key, value):
        self.key = key
        self.value = value
        self.prev = None
        self.next = None

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = {}  # Hash map for O(1) lookups
        self.head = Node(0, 0)  # Dummy head
        self.tail = Node(0, 0)  # Dummy tail
        self.head.next = self.tail
        self.tail.prev = self.head

    def _move_to_end(self, node):
        """Move a node to the end (most recently used)"""
        node.prev.next = node.next
        node.next.prev = node.prev
        node.next = self.tail
        node.prev = self.tail.prev
        self.tail.prev = node
        node.next.prev = node

    def _remove_node(self, node):
        """Remove a node from the list"""
        node.prev.next = node.next
        node.next.prev = node.prev

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        node = self.cache[key]
        self._move_to_end(node)
        return node.value

    def put(self, key: int, value: int):
        if key in self.cache:
            node = self.cache[key]
            node.value = value
            self._move_to_end(node)
        else:
            if len(self.cache) >= self.capacity:
                # Remove least recently used (before head)
                lru_node = self.head.next
                self._remove_node(lru_node)
                del self.cache[lru_node.key]
            new_node = Node(key, value)
            self.cache[key] = new_node
            self._move_to_end(new_node)
```

class Node:
    """Node in the doubly linked list"""
    def __init__(self, key, value):
        self.key = key
        self.value = value
        self.prev = None
        self.next = None


class LRUCache:
    """
    LRU Cache implementation using hash map and doubly linked list.
    Both get and put operations run in O(1) average time.
    """
    
    def __init__(self, capacity: int):
        """
        Initialize the cache with given capacity.
        
        Args:
            capacity: Maximum number of items the cache can hold
        """
        self.capacity = capacity
        self.map = {}  # key -> Node mapping for O(1) lookup
        self.head = None  # Most recently used (front)
        self.tail = None  # Least recently used (back)
    
    def _add_to_head(self, node: Node):
        """
        Add a node to the head (most recently used position).
        Time: O(1)
        """
        if self.head is None:
            self.head = self.tail = node
        else:
            node.next = self.head
            self.head.prev = node
            self.head = node
    
    def _remove_node(self, node: Node):
        """
        Remove a node from the doubly linked list.
        Time: O(1)
        """
        if node.prev:
            node.prev.next = node.next
        if node.next:
            node.next.prev = node.prev
        
        if node == self.head:
            self.head = node.next
        if node == self.tail:
            self.tail = node.prev
    
    def _remove_from_map(self, key):
        """
        Remove a key from the hash map.
        Time: O(1)
        """
        if key in self.map:
            del self.map[key]
    
    def _remove_tail(self):
        """
        Remove the least recently used item (tail).
        Time: O(1)
        """
        if self.tail:
            self._remove_node(self.tail)
            self._remove_from_map(self.tail.key)
    
    def get(self, key: int) -> int:
        """
        Get the value associated with the key.
        If the key exists, move it to the front (most recently used).
        If the key doesn't exist, return None.
        
        Args:
            key: The key to look up
            
        Returns:
            The value if the key exists, None otherwise
        Time: O(1)
        """
        if key in self.map:
            node = self.map[key]
            # Move node to front (most recently used)
            self._remove_node(node)
            self._add_to_head(node)
            return node.value
        return None
    
    def put(self, key: int, value: int):
        """
        Insert or update a key-value pair.
        If the cache is full, remove the least recently used item.
        
        Args:
            key: The key to insert/update
            value: The value to associate with the key
        Time: O(1)
        """
        if key in self.map:
            # Key exists, update value and move to front
            node = self.map[key]
            self._remove_node(node)
            self._add_to_head(node)
            node.value = value
        else:
            # Key doesn't exist
            if len(self.map) >= self.capacity:
                # Cache is full, remove least recently used
                self._remove_tail()
            
            node = Node(key, value)
            self._add_to_head(node)
            self.map[key] = node

 # Pattern Analysis

## Given Mappings

**ABC → DEF**
- A (1) → D (4) = +3
- B (2) → E (5) = +3
- C (3) → F (6) = +3

**XYZ → ABC**
- X (24) → A (1) = +3 (wraps around)
- Y (25) → B (2) = +3 (wraps around)
- Z (26) → C (3) = +3 (wraps around)

## The Rule

Each letter is **shifted forward by 3 positions** in the alphabet, with wrapping from Z back to A.

## Solution

**MNO → PQR**

- M (13) + 3 = P (16)
- N (14) + 3 = Q (17)
- O (15) + 3 = R (18)
